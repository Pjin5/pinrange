Calibrate this phone

For the best accuracy, calibrate against an actual flagstick at a laser-measured distance. The assumed pin height then divides out of both calibration and measurement, so any error in "7 ft" cancels — you're effectively calibrating distance directly.

  1. Stand a laser-measured (or carefully paced) distance from a real flagstick — ideally the practice-green pin, same height as today's course pins.
  2. Lock the frame, hit ✨ Snap, fine-tune TOP & BOTTOM.
  3. Enter that pin's height + the true distance, then Save.
